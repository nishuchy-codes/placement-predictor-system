# Placement Prediction System

This is a Machine Learning project that predicts whether a student will be placed or not based on their CGPA and IQ.

## Project Features
*Flask Web App*: Interactive user interface.
*ML Model*: Logistic Regression trained on placement data.
* *Responsive Design*: Works on mobile and desktop.

## Tech Stack
*Python* (Pandas, Scikit-learn, Flask)
*HTML/CSS*
